Lights Out

Lights Out is an electronic game released by Tiger Electronics in 1995. The game originally consists of a 5 by 5 grid of lights. When the game starts, a random number or a stored pattern of these lights is switched on. Pressing any of the lights will toggle it and the adjacent lights. The goal of the puzzle is to switch all the lights on or off, preferably in as few button presses as possible.

This project is a mobile-friendly JavaScript implementation of the game developed for the course "Mobile Internet Content".
